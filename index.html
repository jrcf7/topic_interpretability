How this session works
  1. You will be shown several topic hierarchies generated from the same scientific corpus, each with a different number of topics (N).
  2. For each hierarchy, rate three dimensions on a 1–7 scale: Coverage, Granularity, and Interpretability.
  3. You can freely navigate back and forth between hierarchies using the Back button or the quick-jump buttons at the bottom of each page. Feel free to revise your ratings after seeing others.
  4. After rating all hierarchies, you will be shown the best-scoring hierarchy and invited to optimise it: you can add keywords to existing topics or add new topics entirely.
  5. At the end, download the results as a JSON file and send it to Karen Meersmans (karen.meersmans@ec.europa.eu).
What you are rating
CoverageDoes the set of topics account for all major themes? Are important areas missing?
GranularityIs the level of detail appropriate — neither too broad nor too fine-grained?
InterpretabilityAre labels and descriptions clear? Could you explain each topic to a colleague?
